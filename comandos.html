Comando Função Sintaxe
GIT INIT criar um repositório, deve ser digitado na pasta onde se quer criar git init
GIT STATUS verificar o status atual do repositório git status
GIT ADD Adiciona os arquivos ao estado preparado git add (nome do arquivo) ou git add *.* (neste caso adiciona todos os arquivos)
GIT COMMIT salva no repositório os arquivos que estavam no estado preparado, levando-os para o estado consolidado git commit -m (mensagem identificando quais as alterações realizadas)
GIT LOG mostra o registro de todos os commits já efetuados git log
GIT CHECKOUT retorna um arquivo ou todo o repositório à qualquer commit anterior git checkout (hash do commit) (nome do arquivo – opcional)
GIT BRANCH cria uma nova ramificação do repositório git branch (nome do branch)
GIT MERGE fundir dois branches em um git merge (nome do branch secundário)
